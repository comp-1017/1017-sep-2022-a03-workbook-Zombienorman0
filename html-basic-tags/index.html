Paragraph Tag

The paragraph tag is a block tag. A block tag begins a new line when the content is displayed by the browser. The browser does NOT display the content in a physical style of how it has been entered with the code in the html file.

Add extension live server for go live execution within visual studio code

  1. open VSC and select your extension icon
  2. select (if seen) or search for live server in the extension list
  3. press the install button
  4. after the install, check blue status line right corener bottom for the button text go live
  5. to use go live:
    • open an html file
    • make any adjustments to the page
    • save the changes
    • press go live
